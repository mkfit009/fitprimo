Disponibile solo con GitHub Pro (a pagamento) per usare GitHub Pages da repo privati.

  • Vai al tuo repository su GitHub

  • Clicca su Settings scorri in basso a "Danger Zone"

2. Proteggi il sito con una password tramite JavaScript (accesso soft)

Puoi aggiungere una pagina di login base in HTML/JS. Esempio:

html
<script>
  const password = prompt("Inserisci la password:");
  if (password !== "fit123") {
    document.body.innerHTML = "<h1>Accesso negato</h1>";
  }
</script>

Semplice ma NON sicuro (visibile nel codice sorgente). Va bene per:

  • Progetti personali

  • Evitare accessi casuali
